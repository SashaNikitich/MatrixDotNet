    Access to matrix elements overview.

    There are many approaches in Matrix DotNet receive element or vector of matrix. So let's consider how we can to get element of matrix.

    using System.Text;
using MatrixDotNet;

namespace Samples.logs.AccessToElements
{
    public class AccessToElementsDocs
    {
        public static void Run()
        {

            // Initialize matrix.
            Matrix<int> matrix = new int[3,3]
            {
                {1, 2, 3},
                {4, 5, 6},
                {7, 8, 9}
            };

            // Approaches for obtaining elements.

            // simple way for receive element.
            int first = matrix[1,2];

            // gets element by State.Row
            int second = matrix[1, 1, State.Row]; 

            // gets element by State.Column
            int third = matrix[0, 2, State.Column];
        }
    }
}

    Output

    Matrix: 
  1  |  2  |  3  |
  4  |  5  |  6  |
  7  |  8  |  9  |

first case: 6
second case gets element by row: 5
third case gets element by column: 7

    As you can see the are difference ways to access of matrix and no difference in performance. you can use way which you have more prefer.

    Also, you can get Vector of matrix by row or column, take a look on below code.

    using System.Text;
using MatrixDotNet;
using MatrixDotNet.Extensions;
using MatrixDotNet.Extensions.Builder;
using MatrixDotNet.Vectorization;

namespace Samples.logs.AccessToVector
{
    public class AccessToVectorDocs
    {
        public static void Run()
        {

            // initialize random matrix.
            Matrix<int> matrix = BuildMatrix.RandomInt(4, 4, -10, 10);

            // gets vector of matrix by row index.
            Vector<int> arr1 = matrix[0, State.Row];

            // gets vector of matrix by column index
            Vector<int> arr2 = matrix[0, State.Column];

            // gets vector via method
            Vector<int> arr3 = matrix.GetColumn(1);
            Vector<int> arr4 = matrix.GetRow(2);

            // assign vector to matrix by column and row.
            matrix[0, State.Column] = arr3.Array;
            matrix[1, State.Row] = arr4.Array;
        }
    }
}

    Thus, the same like in access of element can gets vector.

    Output

    gets vector by row index: <6,7,-1,3>

gets vector by column index: <6,4,2,9>

gets vector by column index via method: <7,-2,-3,1>

gets vector by row index via method: <2,-3,-5,-9>

matrix after conversion: 

   7  |   7  |  -1  |   3  |
   2  |  -3  |  -5  |  -9  |
  -3  |  -3  |  -5  |  -9  |
   1  |   1  |   4  |   5  |
